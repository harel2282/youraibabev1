# Your AI Babe — חיבור לגרוק (Grok) דרך Vercel

המדריך הזה לוקח אותך מאפס עד אפליקציה חיה שבה כל בחורה עונה באמת, עם אישיות וזיכרון משלה.
אין צורך בידע בתכנות ואין צורך בטרמינל — הכל בלחיצות בדפדפן.

---

## מה יש בתיקייה הזו

- `index.html` — האפליקציה עצמה (מה שרואים בטלפון).
- `api/chat.js` — "התיבה הנעולה": רצה בשרת של Vercel, מחזיקה את הסיסמה הסודית של גרוק ומדברת איתו. הסיסמה **לעולם לא נשלחת לדפדפן**.
- `package.json` — קובץ הגדרות קטן (לא נוגעים בו).
- `README.md` — הקובץ הזה.

> **חשוב להבין:** ה‑UI של האפליקציה יעבוד בכל מקום, אבל **הצ'אט עם ה‑AI יתחיל לעבוד רק אחרי שמעלים לאתר (Vercel)** — כי שם נמצאת "התיבה הנעולה". אם תפתח את `index.html` סתם במחשב, הבחורות יענו במשפטים כלליים קבועים (נפילה חיננית), לא ב‑AI אמיתי. זה נורמלי.

---

## שלב 1 — להשיג מפתח (API key) מגרוק

1. היכנס ל‑https://x.ai ופתח חשבון.
2. עבור לאזור ה‑API (Developers / API Keys) ולחץ ליצירת מפתח חדש (**Create API Key**).
3. העתק את המפתח ושמור אותו במקום בטוח (זה כמו סיסמה — לא לשתף עם אף אחד).
4. שים לב: שימוש ב‑API עולה כסף לפי שימוש (משלמים ל‑xAI). לשיחה אישית זה גרושים, אבל צריך אמצעי תשלום / קרדיט בחשבון. בדוק בקונסול אם יש לך קרדיט חינם פעיל.

---

## שלב 2 — להעלות את הקבצים ל‑GitHub (בלי טרמינל)

Vercel אוהב לקרוא את הקוד מ‑GitHub. זה קל:

1. פתח חשבון חינם ב‑https://github.com
2. למעלה מימין לחץ על **+** → **New repository**.
3. תן שם (למשל `your-ai-babe`), השאר **Private** אם תרצה, ולחץ **Create repository**.
4. במסך החדש לחץ על הקישור **uploading an existing file** (או כפתור **Add file → Upload files**).
5. גרור לתוך הדפדפן את **כל התוכן של התיקייה הזו** — כולל התיקייה `api` עם הקובץ `chat.js` שבתוכה. ודא שהמבנה נשמר: `api/chat.js` חייב להישאר בתוך תיקיית `api`.
6. לחץ **Commit changes**.

> טיפ: אם הגרירה לא שומרת את תיקיית `api`, העלה קודם את הקבצים הראשיים, ואז צור קובץ חדש ב‑GitHub בשם `api/chat.js` והדבק לתוכו את התוכן.

## שלב 3 — לפרוס ב‑Vercel

1. פתח חשבון ב‑https://vercel.com והתחבר עם חשבון ה‑GitHub שלך.
2. לחץ **Add New… → Project**.
3. בחר את ה‑repository שיצרת (`your-ai-babe`) ולחץ **Import**.
4. אל תשנה כלום בהגדרות הבנייה (Vercel מזהה לבד) — לחץ **Deploy**.
5. אחרי דקה תקבל כתובת אתר חיה (משהו כמו `your-ai-babe.vercel.app`).

בשלב הזה האתר עלה, אבל הצ'אט עדיין לא ידבר עם גרוק — חסרה הסיסמה. ממשיכים:

---

## שלב 4 — להכניס את המפתח של גרוק (הכי חשוב)

1. בתוך הפרויקט ב‑Vercel: **Settings → Environment Variables**.
2. הוסף משתנה חדש:
   - **Key (שם):** `XAI_API_KEY`
   - **Value (ערך):** המפתח שהעתקת מ‑xAI בשלב 1.
3. שמור (**Save**).
4. עבור ל‑**Deployments**, פתח את הפריסה האחרונה ולחץ **Redeploy** (כדי שהמפתח ייכנס לתוקף).

עכשיו פתח את כתובת האתר — הבחורות עונות באמת. 🎉

---

## שלב 5 — לפתוח בטלפון כמו אפליקציה

פתח את הכתובת בטלפון. באייפון (Safari) אפשר ללחוץ "שתף" → **הוסף למסך הבית**, וזה ייפתח כמו אפליקציה במסך מלא.

## איך משנים דברים

**לשנות אישיות / שם / גיל / תמונות של בחורה:**
ערוך את `index.html`. בחיפוש (Ctrl/Cmd+F) חפש `const PROFILES` — שם מוגדרות כל הבחורות (שם, גיל, עיסוק, ביו, תחומי עניין). שנה מה שתרצה, שמור, והעלה שוב את הקובץ ל‑GitHub (Vercel יפרוס מחדש אוטומטית).

**לשנות את החוקים / אופי התשובות (מה מותר ומה אסור):**
ערוך את `api/chat.js`. בראש הקובץ יש רשימה בשם `RULES` — כל שורה היא חוק. הוסף, מחק או שנה שורות (למשל "לעולם אל תדברי על פוליטיקה"). שמור והעלה שוב.

**להחליף את מודל גרוק:**
ברירת המחדל היא `grok-4.3`. אם תרצה מודל אחר (זול/מהיר יותר), הוסף ב‑Vercel משתנה סביבה נוסף בשם `GROK_MODEL` עם שם הדגם הרצוי (תמצא את הרשימה בקונסול של xAI), ואז **Redeploy**.

## עלויות ודברים שכדאי לדעת

- **גרוק:** תשלום לפי שימוש ל‑xAI. שיחה אישית עולה גרושים, אבל זה לא חינם לגמרי.
- **Vercel (תוכנית Hobby):** חינם, בלי בעיות "שינה". **אבל** התוכנית החינמית מיועדת לשימוש אישי בלבד — אם יום אחד האפליקציה תייצר הכנסה / תהיה מסחרית, תנאי השימוש דורשים מעבר לתוכנית Pro בתשלום.
- **פרטיות:** אם הפעלת קרדיט חינם של xAI דרך תוכנית שיתוף‑נתונים, ייתכן שהשיחות משמשות אותם לאימון. בדוק בקונסול אם זה רלוונטי לך.

---

## פתרון תקלות

- **הבחורות עונות במשפטים כלליים קבועים, לא ב‑AI:** השרת לא מצליח לדבר עם גרוק. בדוק שהוספת `XAI_API_KEY` ושעשית **Redeploy** אחרי זה.
- **שגיאה 401 / Unauthorized:** המפתח שגוי או הודבק עם רווחים. צור מפתח חדש והדבק שוב ב‑Environment Variables.
- **שגיאה על שם המודל (model not found):** שם הדגם השתנה. היכנס לקונסול של xAI, מצא שם דגם תקין, והגדר אותו במשתנה `GROK_MODEL`.
- **התשובה "נחתכת" באמצע / שגיאת timeout:** הטייר החינמי של Vercel חותך פונקציה אחרי 10 שניות. שמור על תשובות קצרות (החוקים כבר מבקשים זאת), או שדרג ל‑Pro לקבלת 60 שניות.
